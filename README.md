# To-Do-List-Python
O To-Do-List é uma ferramenta simples, mas poderosa, usada para organizar obrigações e metas, ajudando a aumentar a produtividade e a gestão do tempo. Esse projeto consiste em lógica de programação, principalmente a nível de backend, afim de demonstrar operações CRUD e serviços web atráves de rotas dinâmicas utilizando sqlite3 e Flask, respectivamente.

# Tecnologias e Libs Utilizadas
- Python
- Flask 
- Sqlite3
- HTML5

# Estrutura de Arquivos
```bash
.
├── .gitignore          # ignore arquivos e pastas
├── README.md           # esse cara que você está lendo
├── app.py              # Aplicação Flask com Lógica CRUD e rotas
└── requirements.txt    # Lista de dependências necessárias
```

# Requisitos e Passos para Execução
Antes de executar o projeto, certifique-se de ter:

- Python 3.10+
- pip instalado
- Virtualenv (opcional, mas recomendado)

### 1. Ambiente Virtual
```bash
python -m venv venv
source venv/bin/activate    # linux / mac
venv\Scripts\activate       # windows
```

### 2. Dependências
```bash
pip install -r requirements.txt
```

### 3. Executar
```bash
python app.py
```
